# TOKYO

> 見ているようで、見ていない風景。「日常の美術館」を体験する探索ゲームアプリ 🖼️✨

## 📖 概要 (Overview)
「TOKYO」は、日常の何気ない風景を新しい視点で見つめ直すことをコンセプトにした探索ゲームアプリです。
ユーザーが撮影した写真から、AIが自動で物体を切り抜いて問題を生成します。ズームやスワイプを駆使し、元の写真からその場所を探し出します。
当たり前だと思っていた景色の中に隠れた「小さな発見」を見つける体験を通し、日常を少し豊かにする『日常の美術館』を目指しました。

## ✨ 主な機能 (Features)
- **AIによる動的な問題生成**: Core ML と Vision API を用い、写真内の物体を自動検出します。どんな写真でも即座にゲーム化することが可能です。
- **高度な画像切り抜き**: iOS 17の Subject Lifting（前景インスタンス分離）技術を活用し、被写体の背景を透過して美しく切り抜きます。
- **直感的なゲームプレイ**: ピンチアウトでのズームやドラッグでの移動に完全対応。アイテムが見つからない時のヒント機能（黄色の枠線）も搭載しています。
- **コレクションと共有**: 見つけたアイテムをローカル（JSON）に永続化して保存。クリア結果は1枚の美しいデザインカード画像として自動生成され、SNS等へシームレスに共有できます。
- **洗練されたUI/UX**: 上品なGlassmorphism（すりガラス効果）を取り入れたUIや、ゆったりとしたクロスフェードアニメーションにより、没入感のあるエモーショナルな世界観を表現しています。

## 🛠 技術スタック (Tech Stack)
- **Target OS**: iOS 17.0+
- **Language**: Swift 6
- **Frameworks**: SwiftUI, Vision, Core ML, PhotosUI
- **Machine Learning**: YOLOv3TinyFP16 (Object Detection)
- **Data Persistence**: JSONEncoder / JSONDecoder (Local Storage)

## 🎮 遊び方 (How to Play)
1. **風景を切り取る**: ホーム画面からカメラを起動するか、アルバムからお気に入りの写真を選択します。
2. **難易度を決める**: 「Easy (3 items)」「Normal (5 items)」「Hard (7 items)」から探すアイテムの数を選択します。
3. **小さな魔法を探す**: 画面下に表示された切り抜きアイテムと同じ場所を、上の写真からタップして見つけ出します。
4. **保存と共有**: すべて見つけるとクリア！コレクションに名前をつけて保存し、結果のカードをシェアしましょう。
